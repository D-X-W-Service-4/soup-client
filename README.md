# SOUP Client

SOUP은 학생의 학습 상태를 기반으로 수준 테스트, 문제 풀이, 학습 플래너를 제공하는 수학 학습 웹 서비스입니다.

## 주요 기능

- 소셜 로그인 및 온보딩
- 닉네임, 학년, 학기, 학습 단원 설정
- 수준 테스트 진행
- 객관식/단답형 문제 풀이
- 문제 별표 및 복습 기능
- 학습 플래너 조회 및 관리
- 학습 기록 및 테스트 결과 확인

## 기술 스택

- React
- TypeScript
- Vite
- React Router
- Zustand
- Axios
- Tailwind CSS
- Framer Motion
